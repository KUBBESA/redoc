Introducción

Este API esta docoumentado en formato OpenAPI desarrollado por el equipo de desarrollo de Crediviva. Para esta documentación, puedes utilizar el api key special-key para probar los filtros de autorización.

Información de Autenticación

Crediviva ofrece la siguiente forma de autenticación:

  • API Key

api_key

Para este ejemplo, puedes utilizar el special-Key el cual te concede de la autenticación necesaria para consumir los recursos expuestos para las operaciones descritas en el API.

Security Scheme Type: API Key
Header parameter name: Authorization
